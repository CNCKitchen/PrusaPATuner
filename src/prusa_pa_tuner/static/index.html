Diagnostics

Live UDP stats (auto-refreshes every second). If a metric you M331'd is missing or running at ~0 Hz, that name probably doesn't exist on this firmware build — the M331 handler writes "Metric not found" to serial and PrusaLink discards the reply, so the failure is invisible from gcode. Throttle is compile-time (the 4th arg of each METRIC_DEF); M331/M332 can only enable/disable, not re-rate. Power-cycling the printer reverts every metric to its compile-time enable/disable default.
